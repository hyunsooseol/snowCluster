# snowCluster

`snowCluster` is a jamovi module developed and maintained by snowsoft Inc.

It is part of the snowsoft statistical module series for jamovi and provides clustering, classification, time series, dimension reduction, correspondence analysis, factor analysis, and machine learning tools for education, research, and applied data analysis.
